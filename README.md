# Learn Go with Tests

### Links
- https://quii.gitbook.io/learn-go-with-tests/

### Usage
- Create a new module with
  ```sh
  go mod init <mod-name>
  ```
- Run the program with
  ```sh
  go run <file-name>.go
  ```
- Run tests with
  ```sh
  go test -v
  ```
- Run benchmarks with
  ```sh
  go test -bench=.
  ```
